International sites

  • Mono Hispano: The spanish-speaking community of Mono developers and users. They have quite a number of unique documents and tutorials.
  • Mono Poland: Mono site for polish users.
  • Mono Japanese Translation: Mono site Japanese translation pages. Translated sources are maintained at Novell Forge.
  • Netherlands Mono Site: Mono forum in the Netherlands.
  • Mono Brasil: Mono discussion site for brazilian users.
  • Two German Mono Site: mono-projekt.de and mono-project.de: Mono site, forum and mailing lists in German.

User sites

  • Alp Toker's site: He is an active Mono developer (gsirc, platano) and produces very nice Debian packages of Mono.
  • Brian Ritchie's site: Brian is a contributor to Mono's class library (ODBC provider and the database multiplexing assembly), and has built an application server for .NET and Mono. His site contains news, software, as well as his weblog.
