Linux Binary Installation

Using Novell ZENworks Linux Management (formerly known as Red Carpet) with the Mono channel is the prefered way to install Mono for the following Linux distributions: Suse 9.x, SLES 8.x, Red Hat 9.0, Fedora Core 1 and Fedora Core 2. RPMs, Yum repositories (for Fedora Core 1 & 2) and source tarballs are also available for these distributions on our download page.

To Run Mono Applications: If you want to run simple .NET applications, install the packages "mono" and "libgc."

To Run GNOME Applications that use Mono: If you want to run GNOME applications that take advantage of .NET technologies, install the gtk-sharp package.

To Develop Mono Applications: install the MonoDevelop package which will pull in all the required dependencies.

To run ASP.NET Applications: install the XSP server and its dependencies.
