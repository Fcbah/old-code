Class Libraries

Classes in Mono are organized by the assembly that they belong to.

Here is the status of the different assemblies:

  • corlib: The core library.
  • cscompmgd: Managed interface for C# compiler.
  • System: Extra modules
  • System.Xml.
  • System.Configuration.Install.
  • System.Data: Database access functionality (also check the Mono ADO.NET status pages).
  • System.Data.OracleClient.
  • System.Design.
  • System.Drawing.
  • System.Drawing.Design.
  • System.Management.
  • System.Messaging.
  • System.Web: ASP.NET classes, also check the Mono ASP.NET status pages
  • System.Web.Services.
  • Microsoft.VisualBasic: Support runtime for Visual Basic applications
  • System.EnterpriseServices.
  • System.Runtime.Remoting.
  • Windows.Forms: there is also a mailing list available
  • System.Runtime.Serialization.Formatters.Soap.
  • System.Security: Xml Signature Security classes, also check the Cryptography status pages
