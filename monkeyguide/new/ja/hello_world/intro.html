あなたが以前C#を使っていた場合、本章の大部分は馴染みあるものです。 この章では完璧なチュートリアルではなく、単にC#による開発を始めるためのものです。

平凡な例

// Hello World! : hello.cs

using System;

class Hello {
    static void Main() {
        Console.WriteLine ("Hello, World!");
    }
}
コンパイルするために 
mcs hello.cs
と入力してください。 すると、hello.exeというファイルが作成され、これをmonoで実行させることが出来ます。 
mono hello.exe

mcsコンパイラやmonoランタイムについてもっと詳しく知りたい場合は、manpagesを見てください。
