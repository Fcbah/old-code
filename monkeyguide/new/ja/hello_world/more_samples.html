Monoを学ぶために、他の人が書いたコードを読むのは、良い学習の方法かもしれません。 ここでは良い例を使って学びます。

Monoハンドブックコードサンプル

これら全てのサンプルは、/samplesディレクトリに入っています。

全てのサンプルをコンパイルする時は、/samplesディレクトリで"make"と入力するか、 それぞれのディレクトリで"make"してください。
全てのサンプルを紹介します：

// HelloWorld - /hello/hello.cs
// Copyright 2002 by Johannes Roith

namespace Sample.hello {

                ...

}

サンプルアプリケーション

これらのアプリケーションは実際Monoを使うにあたり役に立つコードがあるかもしれません。

Monumeric

  • 物理などの科学分野の学生のためのヘルパライブラリ・プログラム

MPhoto

  • デジタル写真を管理するソフト (cvs上にのみあります)

Phonic

  • Gstreamerを使ったGtk#オーディオプレイヤー,

Platano Multimedia Player

  • GStreamerを使った、Gtk#ビデオ・オーディオプレイヤー

GSIRC

  • Gtk#を使ったクロスプラットフォームIRCクライアント

GnoMencoder

  • DVDリッピングユーティリティー

PolarViewer

  • Polar心拍モニタで記録した運動ファイルのビューア

Jump n Bump menu

  • Jump n Bumpのグラフィカルなフロントエンド

Bomb O Naval

  • 大海戦ボードゲーム

ImageBrowser and ImageViewer

  • イメージツール
