この章を読むには、.NETについての知識を事前に学んでおく必要があります。 Monoがあれば、あなたが知っている.NETの概念だけで作業ができますが、LinuxのGUIで動かすための 新しいライブラリを紹介します。これらは、GNOMEのツール群を使って LinuxのGUIを開発している開発者には良く知られたライブラリですが、 Windows開発者にはなじみが無いと思いますので、学ぶ必要があると思います。これが一覧です。

  • art : 2Dグラフィックライブラリ.
  • atk : アクセシビリティ ツールキット
  • gconf : Windowsのレジストリのような、ユーザ単位で構成ファイルを管理
  • gda : GNOMEデータアクセス
  • gdk : X11ウィンドウシステムとGtkの中間に位置するライブラリ
  • glade : ユーザインターフェイス作成ツール
  • glib : X11システムとのインターフェイス
  • gnome : GTK上のデスクトップ環境ライブラリ。アイコン、フォント、印刷、ドルイド(ウィザード)等を含みます
  • gnomedb :
  • gnomeprint : まだ実装されていません
  • gnometutorial : まだ実装されていません
  • gnomevfs : まだ実装されていません
  • gst : まだ実装されていません
  • gtk : ウィジェットセットです。カレンダー、表、ボタン、メニュー、プログレスバー、ツリーなどを作成するオブジェクトを含みます
  • gtkhtml : HTMLレンダラーです
  • panelapplet : ツールバー／パネルウィジェットを書くものです
  • pango : 国際化に対応しているテキストレンダラです
  • parser : ツール群
  • rsvg : librsvgを使った、スケーラブルベクタグラフィックライブラリです。
