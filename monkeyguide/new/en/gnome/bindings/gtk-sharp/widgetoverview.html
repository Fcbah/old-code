The general steps to creating a widget in Gtk# are:

  • 1. new Gtk.*() - one of various functions to create a new widget. These are all detailed in this section.
  • 2. Connect all signals and events we wish to use to the appropriate handlers.
  • 3. Set the attributes of the widget.
  • 4. Pack the widget into a container using the appropriate call such as Gtk.Container.Add or Gtk.Box.PackStart (or its HBox or VBox incarnation).
  • 5. Show() the widget.
  • 6. Show() lets Gtk# know that we are done setting the attributes of the widget, and it is ready to be displayed. You may also use Hide() to make it disappear again. The order in which you show the widgets is not important, but it's best to show the window last so the whole window pops up at once rather than seeing the individual widgets come up on the screen as they're formed. A window is a widget too; the children of a widget will not be displayed until the window itself is shown using the Show() method.
