If you've worked with C# before, most of this chapter will be familiar to you. This is not intended to be an exhaustive tutorial, merely a place to start.

A trivial example

// Hello World! : hello.cs

using System;

class Hello {
    static void Main() {
        Console.WriteLine ("Hello, World!");
    }
}
To compile, type 
mcs hello.cs
This results in a file hello.exe, that you can execute with mono: 
mono hello.exe

For more information on the mcs compiler and mono runtime, see the manpages.
