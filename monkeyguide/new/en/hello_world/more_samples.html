As you work to learn Mono, reading other people's code can be a vital way of learning. In that spirit, here are some pointers to good learning examples.

Mono Handbook Code Samples

All code samples used have a representation in the /samples directory.

You can compile all samples by typing "make" in /samples or make each directory on it's own.
All examples mention the path like that:

// HelloWorld - /hello/hello.cs
// Copyright 2002 by Johannes Roith

namespace Sample.hello {

                ...

}

Sample Applications

The following applications may have useful code if you'd like to see how to use Mono in the real world.

Monumeric

  • A helper library/program for Physics and other sciences students.

MPhoto

  • Application for managing your digital photos (only in cvs)

Phonic

  • Gtk# audio Player, makes use of Gstreamer

Platano Multimedia Player

  • Gtk# video/audio player, makes use of GStreamer

GSIRC

  • Gtk# cross platform IRC client

GnoMencoder

  • DVD ripping utility

PolarViewer

  • A viewer for exercise files recorded with Polar heartrate monitors.

Jump n Bump menu

  • A Graphical frontend to Jump n Bump

Bomb O Naval

  • A classic board game, where you will fight a great sea battle.

ImageBrowser and ImageViewer

  • Image tools
