This is intended to be a basic introduction to terms used in Mono, with a particular focus on Gtk#.

Accelerator
A shortcut key for a menu pull-down, usually displayed at the right side of the pulldown menu next to a command name.
Delegate
A C# language feature similar to a function pointer. (FIXME)
Font Scale
IM (Input Method)
Mnemonic
A mnemonic is a shortcut key for a GUI element other than a menu item. It is usually presented as an underlined letter in the label for an element.
Modifier
A key or mouse button that can change the meaning of a keypress. Used in the implementation of Accel objects. Gtk# supports 8 different modifier keys and 5 mouse button modifiers; some of the key values are dependent on the configuration of the underlying X server.
