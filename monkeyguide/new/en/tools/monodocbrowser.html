Introduction

Monodoc is the documentation system for Mono, and it encompasses:

  • Browsing existing documentation both with native clients as well as over the web.
  • Tools for maintaining and producing documentation for class libraries and projects.
  • Collaborative mechanism for editing documentation.

Monodoc has various different browser for the documentation:

  • Native GUI client for Linux, MacOS X and Windows (using Gtk#).
  • Native GUI client for MacOS X (using Cocoa#).
  • Web-based interface for publishing documentation.
  • Command-line interface for browsing documentation from the shell command line.

GUI client

To start the doc browser by executing the program monodoc, or select it from your desktop menus.

Command Line Interface

To browse documentation from a shell terminal, you can use the `mod' command. Use it to lookup types, like this:

To get all the types in a namespace:

mod N:System
mod N:Gtk

This is for a given Type information:

mod T:System.Collections.ArrayList
To get all the members of the ArrayList: 
mod T:System.Collections.ArrayList/M
To get one method (copy paste the url from the output in the above): 
mod 'M:System.Collections.ArrayList.ToArray(System.Type)'
