Getting MySQL

MySQL is offered as a zip file, containing an executeable setup.exe. If you go with the default settings, MySQL, the documentation and other tools are installed to c:\mysql\. The installation will take 20 MB.

Installation

On Windows NT/2000/XP MySql will be installed as a service by default. It will start, when you reboot, but you can also run it manually with 
net start mysql
You can stop it with 
net stop mysql

Windows 95/98/ME

Note, that you should never use Windows 9x as server or MySQL on Windows 9x as production environment.

If you want to work with 9x for development, that's OK.

MySQL ships with a small tool, WinMySQLAdmin, that can help you to start the MySQL server manually on Windows NT/2000/XP and Windows 95/98/ME. You can run it with:

c:\mysql\bin\winmysqladmin.exe
After getting username and password, the tool will resist as a tray icon. It will put itself in the Autorun directory and startup the server, when you start your computer, if you click on "my.ini Setup/Create Button in Start Menu".
