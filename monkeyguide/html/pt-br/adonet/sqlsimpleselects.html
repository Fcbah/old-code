Simple Queries

The most simple SELECT query is: 
SELECT * FROM table
It will return all records, the complete table. You could theoretically write code to select the correct data from such a request on the client side, but this enlarges the code to be written and is slow, too. Normally you want to select only some of the columns and some of the records, using conditions. Also SELECT can give back other things than table data.

Counting records

Self-explaining. 
SELECT COUNT * FROM table

SELECTING only some columns

Very often you don't kneed to know all columns of a certain table. It is more efficient to only retrieve those. Just replace * with the columns, that you want to get. (Eventually in brackets) 
SELECT (firstname, lastname) FROM table
