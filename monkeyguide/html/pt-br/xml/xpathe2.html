If the path starts with // then all elements in the document which fulfill following criteria are selected.

Example 1

XPath Expression: Select all elements BBB 
//BBB
XML: 
<?xml version="1.0">
     <AAA>
          <BBB/>
          <CCC/>
          <BBB/>
          <DDD>
               <BBB/>
          </DDD>
          <CCC>
               <DDD>
                    <BBB/>
                    <BBB/>
               </DDD>
          </CCC>
     </AAA>

Example 2

XPath Expression: Select all elements BBB which are children of DDD

//DDD/BBB
XML: 
<?xml version="1.0">
     <AAA>
          <BBB/>
          <CCC/>
          <BBB/>
          <DDD>
               <BBB/>
          </DDD>
          <CCC>
               <DDD>
                    <BBB/>
                    <BBB/>
               </DDD>
          </CCC>
     </AAA>
