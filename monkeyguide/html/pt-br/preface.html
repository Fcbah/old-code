X - The Mono Handbook is a collection of guides and tutorials, that might help you to get started with mono. It covers the mono runtime, the C# language, tools like the debugger, certain parts of the class library, the Database layer ADO.NET, and the Website Framework ASP.NET, as well as Gtk#. It is not meant as a complete documentation.
