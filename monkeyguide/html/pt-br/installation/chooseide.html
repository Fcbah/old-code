Available Choices

Currently there are 3 free, GPLed development enviroments available for use with mono.
  • Emacs
  • Eclipse
  • SharpDevelop

Comparing

Everyone has his own needs, but at least one IDE should fit your needs. I've put together the pros and cons of all of them.

Emacs

+ very powerful
+ multiplatform

- difficult for newcomer

Eclipse

+ powerful
+ multiplatform
+ easy to use
+ widely proven

- needs big and proprietary SUN Java JDK
- very slow on a PII 350 and maybe even on new machines

SharpDevelop

+ powerful
+ easy to use
+ perfectly suited for C#
+ Many designers and assistants.

- only runs on windows.
