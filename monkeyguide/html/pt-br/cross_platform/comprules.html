• Do not use absolute paths (e.g. c:\program files\yourapplication)
  • It's better to use Gtk# than System.Windows.Forms.
  • Do only p/invoke against native libraries if really needed. Make sure, the library is available for Unix and Windows.
  • Do not read Registry keys, trough Microsft.Win32.Registry, that are maintained by Windows. (Run, RunServices, get system information, etc.)
