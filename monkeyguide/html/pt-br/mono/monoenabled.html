Platforms supported

Mono is ported to Windows, MacOS X, FreeBSD and Linux and therefor covers 99% of the desktop market.
On the Server Windows and Linux on x86 should gain 70%. It is however needed to port to SPARC and Solaris.

The mono team has developed a new compilation engine, that is easy to port to different platforms. This new engine also provides a solid framework for code optimization and includes a code pre-compiler to reduce startup time.

Desktops supported

Mono currently has support for:
  • The Gnome desktop enviroment
  • The KDE desktop enviroment
  • The Windows desktop enviroment (in progress)

There is on-going work to support Aqua on MacOS X.
