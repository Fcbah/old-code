The Mono Handbook is for:

  • .NET developers who want to learn how to use the Mono tools
  • Windows developers who want to learn about GNOME 2 application development
  • Developers from UNIX, Linux and Macintosh platforms who want to learn about .NET programming

The Mono handbook does not include an introduction to C#, because there are good C# manuals already available. While this guide aims to be useful to all readers, if you are not already familiar with development procedures or with the C# language you will want to consult Appendix C, Links to other Resources for information not covered here.
