Simple compile

 Mono's VB.NET compiler is fully compatible the the MS compiler (vbc) and has even the same options. 
// Hello World! : hello.vb

Imports System

Module Test
        Sub Main()
                Console.WriteLine ("Hello World!")
        End Sub
End Module
To compile, type 
mbas hello.vb
This results in a file hello.exe, that you can execute with mono: 
mono hello.exe

Specifying the Output Filename

 To compile to another filename: 
mbas /out:someothername.exe hello.vb

Specifying the Target Type

Exe

This is the default and will create a console application. 
mbas /target:exe hello.vb

Winexe

This has no effect currently when used with mono, but this might change when mono integrates with GUI filemanagers like Nautilus. On the MS .NET Framework this causes the runtime to not open a console window, if one is not already opened. 
mbas /target:winexe hello.vb

Library

to compile a library: 
mbas /target:library hello.vb

Module

A module is not a complete application, but only for example one or two classes. Several modules can be compiled to one application. This is useful if you want to compile one application out of a VB.NET module and a C# module. 
mbas /target:module hello.vb
