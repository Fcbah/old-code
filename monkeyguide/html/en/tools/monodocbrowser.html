Introduction

 Mono Documentation Browser, MonoDoc, displays various documentation.
For each kind of documentation a provider is needed. Included with MonoDoc is providers for ECMA documentation and the Mono Handbook.

Start the doc browser by executing the program monodoc.

Command Line Interface

A little featurette that you might not know about in MonoDoc is that it also comes with a command line browser.

The command is called mod, and you can lookup types like this:

To get all the types in a namespace: 

mod N:System    or  
mod N:Gtk
This is for a given Type information: 
mod T:System.Collections.ArrayList
To get all the members of the ArrayList: 
mod T:System.Collections.ArrayList/M
To get one method (copy paste the url from the output in the above): 
mod 'M:System.Collections.ArrayList.ToArray(System.Type)'
