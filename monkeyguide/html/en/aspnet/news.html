The Mono homepage is using an RSS news file to generate the page. The first part to a new mono homepage would be, writing an ASP.NET page to read that news file.

It would also be cool if the mono contributors could more easily add news, so we'll write a set of pages to manage the RSS.

This example shows:

  • Monos XML capabilities
  • ASP.NET WebForms
