Introduction

HTML Controls are in most WebDevelopment Enviroments treated as text and controlled through the Output. This is a big step back compared to Client-side programming. This is the point, where the power of ASP.NET becomes clear.

How would you create a link to the main page from a variable? Traditionally like that:

&lta href="<% Response.Write(mainpage); %>">Link to Main page</a>
This leads to unreadable, and code difficult to maintain. ASP.NET provides a very, very elegant solution.

Welcome Objects!

Like in Gtk#, or System.Windows.Forms many HTML controls are accessible as objects. Their properties can be easily changed. ASP.NET generates pure HTML at runtime. 
void Page_Load(object sender, EventArgs e) 
{ 
   anchor1.HRef = mainpage;
}

<form runat="server">

&lta id="anchor1" runat="server">Link to Main page</a>

</form>
Is there to say anything more? This code looks much, much cleaner and does do exactly the same. Web Pages have finally caught up with Client programming. This chapter will introduce all controls and provide examples.
