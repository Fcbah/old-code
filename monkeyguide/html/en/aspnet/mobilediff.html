Technically the only difference is, that you must inherit a different class and register the mobile namespace. Of course you'll also be only able to use the controls defined in that class.

Differences marked red.

<%@ Page Language="C#" Inherits="System.Web.UI.MobileControls.MobilePage" %>

<%@ Register TagPrefix="mobile" Namespace="System.Web.UI.MobileControls" 
Assembly="System.Web.Mobile" %>
Another big difference is, that you need to make even more separation of code and content. You may not use any HTML tags, because this may not render correctly on cell phones. Also the content must be very abstract and great designs are not possible, but that's not the goal here.
