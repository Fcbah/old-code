Function count() counts the number of selected elements

Example 1

XPath Expression: Select elements which have two children BBB 
//*[count(BBB)=2]
XML: 
<?xml version="1.0">
     <AAA>
          <CCC>
               <BBB/>
               <BBB/>
               <BBB/>
          </CCC>
          <DDD>
               <BBB/>
               <BBB/>
          </DDD>
          <EEE>
               <CCC/>
               <DDD/>
          </EEE>
     </AAA>

Example 2

XPath Expression: Select elements which have 2 children
