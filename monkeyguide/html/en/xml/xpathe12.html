The descendant axis contains the descendants of the context node; a descendant is a child or a child of a child and so on; thus the descendant axis never contains attribute or namespace nodes

Example 1

XPath Expression: Select all descendants of document root and therefore all elements 
/descendant::*
XML: 
<?xml version="1.0">
     <AAA>
          <BBB>
               <DDD>
                    <CCC>
                         <DDD/>
                         <EEE/>
                    </CCC>
               </DDD>
          </BBB>
          <CCC>
               <DDD>
                    <EEE>
                         <DDD>
                              <FFF/>
                         </DDD>
                    </EEE>
               </DDD>
          </CCC>
     </AAA>

Example 2

XPath Expression: Select all descendants of /AAA/BBB

/AAA/BBB/descendant::*
XML:

Example 3
