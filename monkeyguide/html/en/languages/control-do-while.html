Do while

Do while is a iterative statement, just like while or for. It has a behavior very similar to the while one, and thus very similar. Its structur is: 


 do {
	// actions to be taken
	// while the the condition is true
	//
 } while ( condition );
It can be said that the while and do-while statements work the same way. The only difference is that in the while statement, the condition is checked before the sentences are executed (and thus the block can not be executed never), and in the do-while statement, the condition is checked after the first execution of the sentences.

A little sample will show the difference between the two statements. In this program, we will run while and a do-while statements. The condition will be : is the integer n minor than 10? 

//
// ========================================================
// A first sample showing the difference between the use of
// the do-while and while statements
// ========================================================
//

using System;

 public class DoWhileSample {

         public static void Main ()
         {

                 int number = 11;

                 //
                 // WHILE statement - condition false
                 //
		 while (number < 10) {
                         Console.WriteLine (number);
                         number++;
                 }

                 //
                 // DO-WHILE statement - condition false
		 // but sentences executed at least one time
                 //
                 do {
                         Console.WriteLine (number);
                         number++;
                 } while (number < 10);

         }

 }

Next, compile it and run it: 

$ mono do-while.exe
11
$
As seen, the sentences in the do-while loop will be executed for first time even the condition is false (this is caused because the revision of the condition after the block of sentences).

The conclusion is: use the do-while statement when you need the funcionality of the while one, but need to execute the sentence(s) at least one time.
