IEnumerator

IEnumerator defines a set of methods and properties to be implemented. The purpose of a object of this type is to iterate over the objects in a collection, in a read-only fashion. There is not a defined class implementing this interface; however, object implementing the IEnumerable interface must supply a object with the requested features of this interface.

Initially, the IEnumerator object points to one position before the first element, and then the MoveNext () method advances one position, returning true if it is still in the range, and false if it advances after the limits of the collection; the Current property will contain the current object, and will throw an exception if no object at the current position exists. Finally, the method Reset () will set the enumerator to its initial position (one before the first element).

The next example will add some elements to a simple list, the ArrayList class, that containt object in a dynamic way, and then will use the enumerator to iterate over the collection. Note that this enumerator results to be useful when it is neccesary to iterate in a particualr way. If changes would be made during the iteration, an exception will be thrown, cause only read iv avalaible. 

//
// IEnumerableExample.cs
//
// Description:
//      Shows the use of the GetEnumerator () method,
//      implemented following the IEnumerable interface.
//

using System;
using System.Collections;

 public class IEnumerableExample {

         public static void Main (string [] args)
         {

                 //
                 // Class implementing the IEnumerable interface
                 //
                 ArrayList list = new ArrayList ();

                 //
                 // Add ten elements to the list
                 //
                 for (int i = 1; i <= 10; i++)
                         list.Add (i);

                 //
                 // Get the IEnumerator object
                 //
                 IEnumerator enumerator = list.GetEnumerator ();

                 //
                 // Use the enumerator
                 //
                 while (enumerator.MoveNext ())
                         Console.WriteLine ("Next Element: {0}", enumerator.Current);

         }

 }
