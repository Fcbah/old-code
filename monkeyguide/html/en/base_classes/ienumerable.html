IEnumerable

System.Collections.IEnumerable is an interface that all the collections in the library must implement. It only has one member to be implemented, GetEnumerator (), which returns an object of kind Enumerator, that will make possible to iterate over the members of the current collection.

The class System.Collections.ArrayList implements this interface, which will make possible to use it in an example: 

//
// IEnumerableExample.cs
//
// Description:
//      Shows the use of the GetEnumerator () method,
//      implemented following the IEnumerable interface.
//

using System;
using System.Collections;

 public class IEnumerableExample {

         public static void Main (string [] args)
         {

                 //
                 // Class implementing the IEnumerable interface
                 //
                 ArrayList list = new ArrayList ();

                 //
                 // Add ten elements to the list
                 //
                 for (int i = 1; i <= 10; i++)
                         list.Add (i);

                 //
                 // Get the IEnumerator object
                 //
                 IEnumerator enumerator = list.GetEnumerator ();

         }

 }
